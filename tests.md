# Тестирование приложения Random Quote Generator

**Проект:** Random Quote Generator  
**Назначение тестирования:** проверка корректной работы GUI-приложения для генерации случайных цитат.

---

## 1. Проверяемые функции

В ходе тестирования проверялись следующие функции приложения:

- запуск графического интерфейса;
- генерация случайной цитаты;
- отображение текущей цитаты;
- добавление новой цитаты;
- валидация полей ввода;
- сохранение истории в JSON;
- загрузка истории из JSON;
- фильтрация истории по автору;
- фильтрация истории по теме;
- сброс фильтров;
- очистка истории.

---

## 2. Позитивные тесты

| № | Действие | Входные данные | Ожидаемый результат | Результат |
|---|---|---|---|---|
| 1 | Запуск программы | `python main.py` | Открывается окно приложения Random Quote Generator | Успешно |
| 2 | Генерация цитаты | Нажать кнопку «Сгенерировать цитату» | В поле текущей цитаты появляется случайная цитата | Успешно |
| 3 | Добавление новой цитаты | Цитата: `Учиться никогда не поздно`, автор: `Народная мудрость`, тема: `учёба` | Цитата добавляется в историю | Успешно |
| 4 | Сохранение истории | Нажать кнопку «Сохранить историю» | Данные сохраняются в файл `quotes_history.json` | Успешно |
| 5 | Загрузка истории | Перезапустить программу | История загружается из JSON-файла | Успешно |
| 6 | Фильтрация по автору | Автор: `Стив Джобс` | В таблице остаются только цитаты выбранного автора | Успешно |
| 7 | Фильтрация по теме | Тема: `мотивация` | В таблице остаются только цитаты выбранной темы | Успешно |
| 8 | Сброс фильтров | Нажать кнопку «Сбросить фильтры» | Все записи истории снова отображаются в таблице | Успешно |
| 9 | Очистка истории | Нажать «Очистить историю» и подтвердить действие | История удаляется из таблицы и JSON | Успешно |

## 5. Проверка JSON

Пример структуры файла `quotes_history.json`:

```json
[
  {
    "text": "Знание - сила.",
    "author": "Фрэнсис Бэкон",
    "topic": "философия",
    "date": "2026-04-28 11:00:25"
  }
]
```

Проверено:

- данные сохраняются в формате JSON;
- русские символы отображаются корректно;
- история загружается после перезапуска программы;
- фильтрация работает по данным из JSON;
- после очистки истории файл обновляется.
