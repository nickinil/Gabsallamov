## 📝 Описание проекта

**Random Quote Generator** - это графическое приложение на Python для генерации случайных цитат. Приложение позволяет:
- Генерировать случайные цитаты из предустановленного списка
- Добавлять собственные цитаты
- Сохранять историю сгенерированных цитат
- Фильтровать историю по автору и теме
- Сохранять и загружать историю в формате JSON

## 🚀 Требования

- Python 3.7 или выше
- Tkinter (обычно входит в стандартную установку Python)
